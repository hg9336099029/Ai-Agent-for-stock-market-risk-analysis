# AI Stock Risk Analysis Platform - Frontend

React-based frontend for the AI Stock & Portfolio Risk Analysis System.

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Axios** - HTTP client

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Page components
├── services/       # API service layer
├── App.jsx         # Main app component
└── main.jsx        # Entry point
```
